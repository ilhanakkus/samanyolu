# Samanyolu Gezgini (Milky Way Explorer)

Gerçekçi, bilimsel verilere dayanan interaktif bir 3D Samanyolu Galaksisi simülasyonu.

## Özellikler
- **3D Galaksi Modeli**: 250.000 parçacıktan oluşan bilimsel sarmal yapı.
- **Gerçek Gezegen Dokuları**: NASA verilerinden alınan Mars, Dünya, Venüs ve Jüpiter dokuları.
- **Etkileşimli Arama**: Gezegen ve yıldız isimlerine göre (Türkçe/İngilizce) otomatik yakınlaşma.
- **Yolculuk Bilgileri**: Dünya'ya uzaklık ve güncel teknolojiyle tahmini varış süreleri.
- **Premium Arayüz**: Glassmorphism tasarımı ve akıcı GSAP animasyonları.

## Kurulum
```bash
npm install
npm run dev
```

## Teknolojiler
- Three.js
- GSAP
- Vite
- Vanilla CSS
